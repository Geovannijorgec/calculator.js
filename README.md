# Calculadora em Node.js
Este é um projeto de uma calculadora implementada em Node.js. A calculadora realiza operações matemáticas básicas, como adição, subtração, multiplicação e divisão. O objetivo é fornecer um exemplo prático de como desenvolver uma calculadora funcional usando o ambiente de execução Node.js.

## Funcionalidades
- Soma: realiza a operação de adição entre dois números.
- Subtração: realiza a operação de subtração entre dois números.
- Multiplicação: realiza a operação de multiplicação entre dois números.
- Divisão: realiza a operação de divisão entre dois números.
## Pré-requisitos
Node.js instalado no seu computador.
## Como usar
- Faça o download dos arquivos deste projeto para o seu computador.
- Abra o terminal e navegue até o diretório onde os arquivos estão localizados.
- Execute os comandos ˋˋˋ npm install ˋˋˋ  para instalar as bibliotecas.
- Execute os comandos ˋˋˋ npm run calc ˋˋˋ  para executar a calculadora.
- Siga as instruções no terminal para inserir os números e escolher a operação desejada.
- O resultado será exibido no terminal.
